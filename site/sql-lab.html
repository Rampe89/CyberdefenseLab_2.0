SQL Injection

Diese Seite simuliert eine unsauber gebaute Datenbankabfrage. Ziel ist nicht, „echte Systeme“ anzugreifen, sondern zu verstehen, warum ungeprüfte Eingaben problematisch sind.

Was passiert hier?
Das System baut eine Abfrage zusammen und setzt deinen Namen direkt hinein.
SELECT * FROM users WHERE username = 'max';

Deine Aufgabe: Probiere aus, wie sich die Abfrage verändert, wenn du nicht nur einen normalen Namen eingibst. Achte besonders auf Zeichen wie '.

Status: normale Abfrage
{INPUT_CAN_CHANGE_THE_QUERY}

Merksatz: Wenn Eingaben ungeprüft in eine Abfrage übernommen werden, kann sich die Bedeutung der Abfrage verändern.
