Was sind Hashes und wie funktionieren sie?

Ein Hash ist das Ergebnis einer kryptografischen Hashfunktion. Diese nimmt eine Eingabe, zum Beispiel ein Passwort, und berechnet daraus eine Zeichenkette mit fester Länge.

Hashfunktionen sind Einwegfunktionen: Aus dem Hash soll das ursprüngliche Passwort nicht einfach zurückberechnet werden können. Schwache Passwörter können aber über Wörterlisten wiedergefunden werden.

Schwachstelle: schwache Passwörter

Wenn ein Passwort sehr häufig verwendet wird, kann ein Angreifer eine Liste bekannter Passwörter hashen und die Ergebnisse vergleichen. Genau das simuliert der lokale Passwortcracker unten. Es werden keine Daten an externe Webseiten gesendet.

Aufgabe

Finde die ursprünglichen Passwörter der folgenden MD5-Hashes. Nutze dafür den lokalen Passwortcracker auf dieser Seite.

1. 5f4dcc3b5aa765d61d8327deb882cf99
2. 098f6bcd4621d373cade4e832627b4f6

Lokaler Hash-Lookup

Kopiere einen oder mehrere MD5-Hashes aus der Aufgabe in das Feld. Der Lookup gleicht die Eingabe direkt im Browser mit einer kleinen lokalen Wörterliste ab. Es werden keine Daten an externe Webseiten gesendet.


Hinweis: Dieser lokale Lookup funktioniert nur mit den bewusst einfachen Hashes aus der Übung.
