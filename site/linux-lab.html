Linux Grundkurs – Navigation im Terminal

Dieses Zusatz-Lab setzt einen anderen Schwerpunkt als die klassischen Web-Labs. Hier geht es nicht um einen einzelnen Angriff, sondern um grundlegendes Werkzeugverständnis. Wer sich mit Cybersecurity beschäftigt, stößt sehr schnell auf Linux-Systeme, Verzeichnisse, Konfigurationsdateien und die Arbeit im Terminal.

Das Lab ist bewusst simuliert. Es geht nicht darum, ein echtes System zu bedienen, sondern die Grundlogik von ls, cd, pwd, cat und clear praktisch zu erleben. Nach erfolgreichem Abschluss wird der Expert Mode freigeschaltet.

Ablauf

  1. Verschaffe dir einen Überblick über das aktuelle Verzeichnis.
  2. Bewege dich gezielt durch die Struktur.
  3. Prüfe auch versteckte Inhalte.
  4. Lies Hinweise in Dateien statt nur blind herumzuklicken.
  5. Finde die Flag und schalte damit den Expert Mode frei.

Legende

  • ls → Inhalte anzeigen
  • ls -a → auch versteckte Einträge anzeigen
  • cd NAME → Verzeichnis wechseln
  • cd .. → eine Ebene zurück
  • pwd → aktuellen Pfad anzeigen
  • cat DATEI → Dateiinhalt anzeigen
  • clear → Terminal leeren
  • ↑ / ↓ → letzte Befehle durchgehen

Hinweise

Das Ziel ist nicht, möglichst viele Befehle zu kennen, sondern die Logik zu verstehen.

Wenn du festhängst, beginne wieder mit pwd und ls.

Nicht alles ist sofort sichtbar. ls -a ist hier wichtiger als auf den ersten Blick gedacht.

Ein paar kleine Easter Eggs gibt es ebenfalls. Sie helfen nicht direkt, zeigen aber typische Terminal-Gewohnheiten.
