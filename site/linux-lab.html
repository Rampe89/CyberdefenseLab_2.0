Linux Grundkurs – Navigation im Terminal

Dieses Zusatz-Lab setzt einen anderen Schwerpunkt als die klassischen Web-Labs. Hier geht es nicht um einen einzelnen Angriff, sondern um ein grundlegendes Werkzeugverständnis. Wer sich mit Cybersecurity beschäftigt, stößt sehr schnell auf Linux-Systeme, Verzeichnisse, Konfigurationsdateien und die Arbeit im Terminal. Gerade deshalb ist es sinnvoll, dass die Schülerinnen und Schüler zumindest die Grundlogik von ls, cd, pwd und cat einmal praktisch erleben.

Das Lab ist bewusst simuliert. Es geht nicht darum, ein echtes System zu bedienen, sondern ein erstes Gefühl dafür zu entwickeln, wie man sich in einer Dateistruktur bewegt, Hinweise liest und Schritt für Schritt eine gesuchte Datei findet.

Fake Bash

Legende

  • ls → Inhalte anzeigen
  • ls -a → auch versteckte Einträge anzeigen
  • cd NAME → Verzeichnis wechseln
  • cd .. → eine Ebene zurück
  • pwd → aktuellen Pfad anzeigen
  • cat DATEI → Dateiinhalt anzeigen

Hinweise

Das Ziel ist nicht, viele Befehle zu kennen, sondern die Logik zu verstehen.

Nicht alles ist sofort sichtbar. Schau deshalb genau hin und lies Hinweise in Dateien wirklich mit.

Ein paar kleine Easter Eggs gibt es auch. Sie helfen nicht direkt, sorgen aber hoffentlich für ein kurzes Grinsen.
